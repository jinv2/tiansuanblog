专属行业AI智能体使用说明（音乐 / 影视）

欢迎使用我们的音乐与影视行业AI智能体。本说明面向非技术客户，帮助您快速上手，获得专业创作建议和行业参考。

1. 打开智能体页面

访问以下页面，开始使用您的专属行业AI智能体：

https://industryai2.netlify.app/

2. 选择行业

在页面上，您会看到“选择行业”下拉框，支持如下选项：

  • 音乐：生成旋律、和声、混音、编曲、剪辑风格等专业建议。
  • 影视：生成分镜、剪辑顺序、剧情优化方案等建议。

3. 输入您的创作需求

在文本框中填写您的具体需求，例如：

  • 音乐：请生成一段主旋律，并自动补全和声与配器。
  • 影视：短片剧情是侦探寻找线索，建议如何调整场景顺序，增强悬念感。

4. 点击“生成专业建议”

智能体会基于您的输入和行业知识，生成详细建议，帮助您创作或优化作品。

5. 查看与应用结果

生成结果会显示在页面下方，您可以：

  1. 复制文本用于参考或创作
  2. 结合项目实际进行修改优化
  3. 多次输入不同需求获取更多建议

注意事项

  • 智能体内部使用 DeepSeek API 完成运算，无需客户配置或技术操作。
  • 输入尽量清晰具体，越具体得到的建议越精准。
  • 智能体为辅助创作工具，建议结合人工判断使用。
