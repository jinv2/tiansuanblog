DeepSeek 专属行业AI智能体使用说明

感谢您使用 DeepSeek 提供的专属行业AI智能体。本说明面向非技术客户，帮助您快速上手，享受智能体带来的专业建议和行业效率提升。

1. 打开智能体页面

访问以下页面，开始使用您的专属行业AI智能体：

https://industryai2.netlify.app/

2. 选择行业

在页面上，您会看到“选择行业”下拉框，支持如下选项：

  • 音乐：用于生成旋律、和声、混音、剪辑风格等专业建议。
  • 影视：用于生成分镜、剪辑顺序、剧情优化等建议。

3. 输入您的专业需求

在文本框中填写您的具体需求，例如：

  • 音乐：请生成一段主旋律，并自动补全和声和配器。
  • 影视：我的短片剧情是侦探寻找线索，建议如何调整场景顺序，使观众更有悬念。

4. 点击“生成专业建议”

智能体会基于您的输入和行业知识，生成详细的专业建议。整个过程无需安装软件或进行复杂配置。

5. 查看结果与应用

生成的结果会直接显示在页面下方。您可以：

  1. 复制文本并用于创作参考
  2. 结合实际项目进行修改和优化
  3. 重复输入不同需求，获取更多建议

注意事项

  • 智能体使用您自己的 DeepSeek API Key，不会暴露您的数据。
  • 请确保输入清晰具体，越具体的需求得到的建议越精准。
  • 本工具为辅助决策工具，建议与人工判断结合使用。
