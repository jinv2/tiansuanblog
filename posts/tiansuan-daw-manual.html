天算 DAW 使用手册

版本： v4.3 (DeepSeek Engine)  |  发布日期： 2026年1月22日  |  开发者： 天算 AI 科技研发实验室

1. 产品概述

天算 DAW 是一款专为专业音乐制作人打造的 Web-based AI 宿主软件。与传统网页转换工具不同，它采用了 BYOK (Bring Your Own Key) 架构，底层直连 DeepSeek-V3 推理引擎，能够理解复杂的音乐织体，并输出工业级标准的 MusicXML 乐谱。

2. 快速开始 (Quick Start)

第一步：连接推理引擎

由于本系统采用 Serverless 架构，您需要连接 AI 大脑：

  1. 点击界面右上角的 CONNECT API 按钮。
  2. 在弹出的配置窗口中，输入您的 DeepSeek API Key (sk-xxxxxxxx)。
  3. 点击 连接 (Connect)。
  4. 状态确认：当顶部状态栏显示绿色的 ONLINE 时，引擎已就绪。

第二步：导入音频

在左侧的 INSPECTOR (检查器) 面板中：

  • 找到 Audio Source 区域。
  • 点击虚线框或直接将音频文件拖入框内。
  • 支持格式：MP3, WAV, FLAC (建议使用 44.1kHz 以上的高品质音频)。

第三步：选择输出格式

在 Output Format 区域选择目标乐谱类型：

  • Orchestral Score (五线谱)： 适合交响乐、钢琴独奏或复杂编曲分析。
  • Jianpu / Numeric (简谱)： 适合流行音乐或快速旋律记录。

第四步：AI 转换

一切准备就绪后：

  1. 点击顶部波形下方的 DeepSeek Transform 按钮。
  2. 等待推理： 系统会将音频特征发送给 DeepSeek V3 进行逻辑分析（约 10-30 秒）。
  3. 查看结果： 转换完成的乐谱将直接显示在右侧主工作区。

3. 常见问题 (FAQ)

Q: API Key 安全吗？
A: 绝对安全。Key 仅存储在您的本地浏览器 (Local Storage)，绝不上传至任何第三方服务器。

Q: 界面左上角的 Logo 是什么？
A: 那是 TianSuan Engine v4.3 的运行标识。如果您看到它，说明您正在使用最新的稳定版本。
