灵愿简谱 Pro v28.56 使用手册

发布日期：2026-01-25 | 版本状态：Final Stable

灵愿简谱 Pro (Samsara Muse Pro) 是一款基于 Web 端、深度集成物理逻辑排版算法的专家级打谱软件。v28.56 版本作为“物理纠偏”系列的终结版，完美解决了 Web 打谱中常见的间距不均与符号冲突问题。

1. 核心排版技术：物理领土占位

不同于传统的流式排版，v28.56 采用了领土均分算法：

  • 绝对均分：每一小节被逻辑切分为四个相等的“拍位领土”。无论音符多寡，均会自动对齐所属拍位的几何中心。
  • 延时杠 (-) 物理占位：全音符与二分音符的延时杠不再是字符，而是独立的物理对象，确保在 4/4 拍中占据完整的节拍领土。
  • 跨拍死锁：减时线（下划线）在跨越拍位边界时会自动物理断开，确保节奏组层次清晰，告别“一根长线到底”的错误排版。

2. 操作指南

2.1 录入流程

1. 设置元数据：在顶部 Header 区域直接修改标题、副标题、调号（1=F）与词曲作者。
2. 选择时值：在左侧面板点击对应按钮（全音、二分、四分等）。
3. 修饰音符：若需录入附点音符或带有高低音点的音符，请先点亮对应的修饰模块按钮（蓝色表示激活），再点击数字键盘录入。

2.2 版面微调

利用左侧的 “布版控制” 卡片：

  • 调整 单行小节 改变谱面紧凑度。
  • 调整 物理行距 控制纵向呼吸感。
  • 调整 起始高度 为页面顶部留出足够的歌词或插图空间。

3. v28.56 关键补丁说明

  • UI 补全：物理归位了高低音独立修饰模块与 MIDI 导出按钮。
  • 渲染纠偏：修复了 16 分音符减时线长度冲突，向内缩进 4px 实现视觉平衡。
  • 元数据分行：调号/拍号与词曲作者信息正式采用双行专业对齐布局。

4. 导出与存储

系统支持 .tsmuse 工程格式导出，您可以随时保存工作进度并在任意设备通过“导入”功能恢复录入。成品支持 高清切片导出，直接适配打印标准。
