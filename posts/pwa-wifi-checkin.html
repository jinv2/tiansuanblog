PWA Wifi Checkin：极简主义的远程办公打卡器

发布于 2026-01-09 | 分类: Tool Release

对于分布式团队来说，证明“我在线”往往需要繁琐的 VPN 或特定软件。我们认为，基于地理位置和网络的验证应当是无感的。

功能特点

Wifi Checkin 是一个极其轻量的 PWA 应用，大小仅 200KB。

  • SSID 嗅探：自动识别当前连接的 Wi-Fi 名称（部分 Android 设备支持）。
  • IP 地理围栏：基于公网 IP 判断是否处于“办公白名单”区域。
  • 一键打卡：点击按钮，生成包含时间戳、IP、位置签名的加密 Token。
  • 隐私保护：不上传 GPS 轨迹，仅在打卡瞬间比对位置。

该工具现已集成至 Claritas Terminal 系统中。
